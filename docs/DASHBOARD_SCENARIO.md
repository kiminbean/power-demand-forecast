## 1. 현재 수요

**표시 예시**: 689.0 MW (예비율 30.1%)

### 시나리오: 출력제한(Curtailment) 대응

```
상황: 제주도 현재 수요 689 MW, 재생에너지 출력 250 MW
      예비율이 30%를 초과하여 출력제한 발령 예상

대응:
├─ 발전사업자: "수요가 낮아 출력제한 가능성 높음"
│              → ESS 충전으로 전환하여 수익 확보
│
├─ 트레이더: "수요 저조로 SMP 하락 예상"
│            → 실시간 시장(RTM) 판매량 조절
│
└─ 계통운영자: "예비력 과잉 상태"
               → 재생에너지 출력제한 지시 준비
```

### 비즈니스 가치
- 출력제한 전 선제 대응으로 **손실 최소화**
- 예비율 기반 운영 전략 수립

### 임계값 기준

| 예비율 | 상태 | 대응 |
|--------|------|------|
| < 10% | 🔴 위험 | 예비력 투입, 부하차단 준비 |
| 10-15% | 🟡 주의 | 발전기 대기, 모니터링 강화 |
| 15-25% | 🟢 정상 | 정상 운영 |
| > 30% | 🟠 과잉 | 출력제한 검토 |

## 2. 현재 SMP - 제주

**표시 예시**: 693.5원/kWh (+0.0%)

### 시나리오: 최적 입찰 전략 수립

```
상황: 제주 SMP 693.5원 (평균 대비 +15% 고가)
      익일 하루전시장(DAM) 입찰 마감 10:00

분석:
├─ 고가 원인: 연료비 상승 + HVDC 송전제한
│
├─ 태양광 발전사업자:
│   "SMP 고가 구간(09:00-15:00)에 최대 출력 집중"
│   → 예상 추가 수익: 1.2억원/일
│
└─ ESS 운영자:
    "야간 저가(50원) 충전 → 주간 고가(700원) 방전"
    → 차익: 650원/kWh × 50MWh = 3,250만원/일
```

### 비즈니스 가치
- SMP 예측 기반 **수익 극대화 입찰 전략**
- 시간대별 가격 차익 거래(Arbitrage)

### SMP 수준별 전략

| SMP 범위 | 상태 | 전략 |
|----------|------|------|
| < 80원 | 저가 | ESS 충전, 발전 최소화 |
| 80-150원 | 보통 | 정상 운영 |
| 150-300원 | 고가 | 최대 출력, ESS 방전 |
| > 300원 | 초고가 | 전량 판매, 긴급 대응 |

## 3. 재생에너지 비율

**표시 예시**: 8.5% (태양광+풍력)

### 시나리오: 계통 안정성 모니터링

```
상황: 재생에너지 비율 8.5% (정상 범위)
      ※ 제주도 위험 기준: 50% 초과 시 불안정

모니터링 시나리오:
├─ 현재 (8.5%): ✅ 안정
│   "동기발전기 충분, 주파수 안정"
│
├─ 주간 피크 (35-45%): ⚠️ 주의
│   "관성력 저하, 주파수 변동 증가 가능"
│   → 동기조상기 가동 준비
│
└─ 봄철 오후 (50%+): 🔴 위험
    "출력제한 필수, HVDC 역송전 필요"
    → 재생에너지 출력제한 발령
```

### 비즈니스 가치
- 계통 붕괴 방지 및 **안정적 전력 공급**
- 출력제한 사전 예측

### 제주도 재생에너지 비율 기준

| 비율 | 상태 | 계통 영향 | 대응 |
|------|------|----------|------|
| < 20% | 🟢 안정 | 관성력 충분 | 정상 운영 |
| 20-35% | 🟡 관심 | 변동성 증가 | 모니터링 강화 |
| 35-50% | 🟠 주의 | 주파수 불안정 | 동기조상기 가동 |
| > 50% | 🔴 위험 | 계통 불안정 | 출력제한 발령 |

## 4. 계통 주파수

**표시 예시**: 60.02 Hz (정상)

### 시나리오: 실시간 수급 균형 감시

```
기준: 한국전력계통 표준 60.00 Hz ± 0.2 Hz

주파수 상태별 대응:
├─ 60.02 Hz (현재): ✅ 정상
│   "공급 > 수요 (미세 초과)"
│   → 정상 운영
│
├─ 59.90 Hz: ⚠️ 저주파수 경보
│   "수요 > 공급 (공급 부족)"
│   → 예비력 투입, 부하차단 준비
│
├─ 60.20 Hz: ⚠️ 고주파수 경보
│   "공급 >> 수요 (과잉 공급)"
│   → 발전기 출력 감소, 재생에너지 제한
│
└─ 59.70 Hz 이하: 🔴 비상
    "계통 붕괴 위험"
    → 자동 부하차단(UFLS) 작동
```

### 비즈니스 가치
- **블랙아웃 방지** 및 전력품질 유지
- 실시간 수급 균형 상태 파악

### 주파수 임계값

| 주파수 범위 | 상태 | 의미 | 자동 대응 |
|------------|------|------|----------|
| 60.20+ Hz | 🟠 고주파수 | 공급 과잉 | 발전기 출력 감소 |
| 59.97-60.03 Hz | 🟢 정상 | 균형 상태 | - |
| 59.90-59.97 Hz | 🟡 주의 | 공급 부족 | 예비력 대기 |
| 59.70-59.90 Hz | 🟠 경보 | 심각한 부족 | 예비력 투입 |
| < 59.70 Hz | 🔴 비상 | 계통 위험 | UFLS 작동 |

## 5. 기상 현황

**표시 예시**: 9°C, 풍속 4.6m/s, 맑음

### 시나리오: 재생에너지 출력 예측

```
현재 기상: 기온 9°C, 풍속 4.6m/s, 맑음

발전량 예측 활용:
├─ 태양광 발전사업자:
│   "맑음 + 기온 9°C = 태양광 효율 최적"
│   → 일사량 대비 105% 출력 예상
│   → 하루전시장 입찰량 상향 조정
│
├─ 풍력 발전사업자:
│   "풍속 4.6m/s = 정격출력의 30%"
│   ※ 정격풍속 12m/s 기준
│   → 출력 예측: 146MW × 30% = 44MW
│
└─ ESS 운영자:
    "오후 태양광 피크 시 잉여전력 예상"
    → 14:00-16:00 충전 스케줄 설정
```

### 비즈니스 가치
- 정확한 발전량 예측으로 **입찰 정확도 향상**
- 기상 기반 운영 계획 수립

### 기상-발전량 관계

| 기상 요소 | 영향 발전원 | 관계 |
|----------|-----------|------|
| 일사량 | 태양광 | 선형 비례 |
| 기온 | 태양광 | 저온 시 효율 증가 |
| 풍속 | 풍력 | 3-25m/s 발전 가능 |
| 습도 | 태양광 | 고습도 시 효율 감소 |

### 풍력 출력 곡선 (Power Curve)

| 풍속 | 출력 비율 | 상태 |
|------|----------|------|
| < 3 m/s | 0% | Cut-in 미달 |
| 3-8 m/s | 10-50% | 부분 부하 |
| 8-12 m/s | 50-100% | 상승 구간 |
| 12-25 m/s | 100% | 정격 출력 |
| > 25 m/s | 0% | Cut-out (안전정지) |

## 6. 제주 전력수급 현황 차트

**표시**: 24시간 공급능력/수요/태양광/풍력 시계열 차트

### 시나리오: 시간대별 운영 전략

```
차트 분석 (실측 + 예측):
├─ 06:00-09:00: 수요 상승기
│   "태양광 출력 시작, 수요 급증"
│   → 피크 대비 발전기 기동
│
├─ 09:00-15:00: 태양광 피크
│   "공급과잉 구간, 출력제한 가능"
│   → ESS 충전 최적 시간대
│
├─ 17:00-21:00: 저녁 피크
│   "태양광 종료, 수요 최대"
│   → ESS 방전 + 고가 SMP 수익 구간
│
└─ 실측/예측 경계선:
    "현재 시점 이후는 예측값"
    → 예측 오차 모니터링
```

### 비즈니스 가치
- 시간대별 **최적 운영 의사결정 지원**
- 실측 vs 예측 비교로 예측 모델 정확도 검증

### 시간대별 운영 전략

| 시간대 | 특성 | 태양광 | 풍력 | ESS | SMP |
|--------|------|--------|------|-----|-----|
| 00-06시 | 야간 저수요 | - | 중 | 충전 | 저가 |
| 06-09시 | 아침 상승 | 시작 | 중 | 대기 | 상승 |
| 09-15시 | 주간 피크 | 최대 | 중 | 충전 | 변동 |
| 15-18시 | 오후 하락 | 감소 | 중 | 대기 | 상승 |
| 18-22시 | 저녁 피크 | - | 중 | 방전 | 고가 |
| 22-24시 | 야간 진입 | - | 중 | 충전 | 하락 |

## 7. 발전량 히트맵 표시

**표시**: 제주도 지도 위 발전소 위치 및 발전량 시각화

### 시나리오: 지역별 계통 혼잡 관리

```
히트맵 분석:
├─ 동부 (성산, 구좌): 🟢 풍력 밀집
│   "해상풍력 + 육상풍력 집중"
│   → 동부 변전소 혼잡 시 출력제한 우선
│
├─ 서부 (한경, 대정): 🟡 태양광 + 풍력
│   "복합 발전단지"
│   → 변동성 높음, ESS 필요 지역
│
├─ 중부 (제주시): 🔵 ESS + 연계
│   "HVDC 연계점, 계통 허브"
│   → 전력 흐름 중심점
│
└─ 발전소 크기 = 설비용량
    발전소 색상 = 발전 유형
```

### 비즈니스 가치
- 지역별 계통 혼잡 예측 및 **선제적 출력 조정**
- 송전 제약 구간 식별

### 제주도 주요 연계점

| 지역 | 주요 설비 | 연계 용량 | 혼잡 위험 |
|------|----------|----------|----------|
| 제주시 | HVDC #1 | 200MW | 중 |
| 서귀포 | HVDC #2 | 200MW | 중 |
| 동부 | 풍력단지 | 150MW+ | 고 |
| 서부 | 복합단지 | 100MW+ | 중 |

## 8. 발전소 현황

**표시**: 발전원별 개소 수, 설비용량, 현재 발전량

### 시나리오: 포트폴리오 실시간 모니터링

```
현황 데이터:
├─ 태양광: 7개소 | 48.5MW (발전량: 34.1MW)
│   이용률: 70% ✅ 양호
│
├─ 풍력: 10개소 | 146MW (발전량: 108.5MW)
│   이용률: 74% ✅ 양호
│
└─ ESS: 3개소 | 50MW (충전량: 13.4MW)
    상태: 충전 중 (SOC 27%)

운영 의사결정:
├─ "태양광 이용률 70% → 출력제한 없음 확인"
├─ "풍력 이용률 74% → 양호한 풍황"
└─ "ESS SOC 27% → 오후 피크 방전 가능"
```

### 비즈니스 가치
- 자산별 **실시간 성능 모니터링** 및 이상 감지
- 포트폴리오 전체 수익성 파악

### 이용률 기준

| 발전원 | 우수 | 양호 | 보통 | 저조 |
|--------|------|------|------|------|
| 태양광 | >80% | 60-80% | 40-60% | <40% |
| 풍력 | >40% | 25-40% | 15-25% | <15% |
| ESS | - | - | - | - |

### ESS 상태별 운영

| SOC 범위 | 상태 | 운영 전략 |
|----------|------|----------|
| 0-20% | 🔴 저충전 | 즉시 충전 필요 |
| 20-40% | 🟡 저충전 | 충전 권장 |
| 40-60% | 🟢 적정 | 유연 운영 |
| 60-80% | 🟢 고충전 | 방전 가능 |
| 80-100% | 🟠 만충전 | 방전 필요 |
