# KPX 영업 시나리오

> **제주도 전력 수요 예측 AI 시스템 도입 제안**

---

## 1. 영업 대상 분석

### 1.1 KPX (한국전력거래소) 프로필
- **역할**: 전력시장 운영, 전력계통 운영, 전력수급 계획
- **주요 관심사**:
  - 전력 수급 안정성 확보
  - SMP(계통한계가격) 예측 정확도
  - 예비율 최적화를 통한 비용 절감
  - 재생에너지 확대에 따른 변동성 대응

### 1.2 잠재적 Pain Points
| 문제 | 현재 상황 | 우리 솔루션 |
|------|----------|------------|
| 예측 불확실성 | 기존 통계 모델의 한계 | AI 기반 10.68% MAPE 달성 |
| 피크 수요 관리 | 사후 대응 중심 | 24시간 사전 예측 |
| 설명 부재 | 블랙박스 예측 | XAI 기반 설명 가능 |
| 시나리오 분석 | 수작업 시뮬레이션 | 자동화된 What-if 분석 |

---

## 2. 영업 시나리오 (5단계)

### Phase 1: 도입 (Opening) - 5분

**목표**: 관심 유도 및 문제 공감대 형성

```
"안녕하세요, 저희는 제주도 전력 수요 예측 AI 시스템을 개발했습니다.

최근 제주도의 재생에너지 비중 증가로 전력 수급 예측이
더욱 어려워지고 있다는 점을 알고 있습니다.

특히 태양광 발전의 '덕 커브(Duck Curve)' 현상이
기존 예측 모델의 정확도를 떨어뜨리고 있죠.

저희 시스템은 이 문제를 해결하기 위해 개발되었습니다."
```

**Key Points**:
- 제주도 특수성 언급 (재생에너지, 독립 계통)
- 현재 겪고 있는 문제점 공감
- 솔루션 존재 암시

---

### Phase 2: 문제 진단 (Discovery) - 10분

**목표**: 구체적 니즈 파악 및 문제 심화

**질문 리스트**:
1. "현재 전력 수요 예측의 가장 큰 어려움은 무엇인가요?"
2. "예측 오차로 인한 비용은 연간 얼마나 발생하나요?"
3. "피크 수요 예측 실패 시 어떤 조치를 취하시나요?"
4. "SMP 예측에 대한 현재 시스템의 정확도는 어느 정도인가요?"

**예상 응답 및 대응**:

| 고객 응답 | 우리 대응 |
|----------|----------|
| "피크 시간대 예측이 어려워요" | "24시간 전 피크 예측 가능, MAPE 10.68%" |
| "재생에너지 변동성 대응이 힘들어요" | "기상 데이터 연동 실시간 업데이트" |
| "예측 근거를 알 수 없어요" | "XAI로 모든 예측의 근거 설명" |
| "다양한 시나리오를 테스트하고 싶어요" | "What-if 분석 자동화 지원" |

---

### Phase 3: 솔루션 제시 (Presentation) - 15분

**목표**: 제품 가치 전달

#### 3.1 시스템 개요 (와이어프레임 05번 활용)

```
"저희 시스템은 4가지 핵심 모듈로 구성됩니다:

1. 데이터 수집 모듈
   - EPSIS 실시간 연동
   - 기상청 AMOS 데이터
   - 공공데이터포털 연계

2. AI 예측 엔진
   - LSTM, BiLSTM, TFT 앙상블
   - Quantile Regression으로 불확실성 정량화
   - Walk-forward CV로 검증된 모델

3. XAI 분석 모듈
   - Attention 기반 설명
   - SHAP 피처 중요도
   - 자연어 예측 근거 제공

4. 대시보드
   - 실시간 모니터링
   - 시나리오 시뮬레이션
   - API 연동 지원"
```

#### 3.2 메인 대시보드 데모 (와이어프레임 01번 활용)

```
"이것이 메인 대시보드입니다.

상단에서 현재 수요와 예측 수요를 한눈에 확인할 수 있고,
중앙 그래프에서 24시간 예측과 80% 신뢰구간을 보여줍니다.

우측에서는 실시간 알림을 확인할 수 있어
피크 수요 예상 시 사전 대응이 가능합니다."
```

#### 3.3 SMP 예측 및 입찰 지원 (와이어프레임 02번 활용)

```
"SMP 예측 페이지에서는 48시간 가격 예측을 제공합니다.

특히 Quantile 예측으로 10%, 50%, 90% 분위수를 제공해
입찰 전략 수립에 도움을 드립니다.

또한 최적 입찰가와 낙찰 확률까지 추천해드립니다."
```

#### 3.4 XAI 분석 (와이어프레임 03번 활용)

```
"가장 차별화된 기능인 XAI 분석입니다.

왼쪽의 Attention 히트맵에서 모델이 어떤 시점의 데이터에
주목하는지 시각적으로 확인할 수 있습니다.

오른쪽에서는 각 피처의 기여도를 확인할 수 있어
'왜 이런 예측이 나왔는지' 명확히 설명됩니다.

하단의 자연어 설명은 비전문가도 이해할 수 있는
형태로 예측 근거를 제공합니다."
```

#### 3.5 시나리오 분석 (와이어프레임 04번 활용)

```
"시나리오 분석 페이지에서는 What-if 시뮬레이션이 가능합니다.

폭염, 한파 등 사전 정의된 시나리오를 선택하거나
원하는 조건을 직접 설정할 수 있습니다.

기본 시나리오 대비 수요 변화와 리스크를 실시간으로
확인하여 대응 계획을 수립할 수 있습니다."
```

### Phase 4: 가치 증명 (Value Demonstration) - 10분

**목표**: ROI 및 구체적 효과 제시 (와이어프레임 06번 활용)

#### 4.1 정량적 가치

| 항목 | 현재 추정 | 도입 후 기대 | 개선 효과 |
|------|----------|-------------|----------|
| 예측 MAPE | 15~20% | 10.68% | 5~10%p 개선 |
| 예비율 최적화 | 과잉 예비 | 적정 예비 | 3~5% 절감 가능 |
| 의사결정 시간 | 수시간 | 실시간 | 90% 단축 |
| 시나리오 분석 | 수작업 | 자동화 | 80% 효율화 |

#### 4.2 정성적 가치

1. **투명성**: 모든 예측의 근거를 설명 가능
2. **신뢰성**: 82.5%의 신뢰구간 정확도
3. **확장성**: API 기반으로 기존 시스템과 연동 용이
4. **지속성**: 지속적 모델 업데이트 지원

#### 4.3 Reference

```
"이 모델은 2022-2024년 실제 EPSIS 데이터 26,240건으로
학습 및 검증되었습니다.

Walk-forward 5-fold 교차검증을 통해 과적합 없이
일반화 성능이 검증되었습니다.

관련 연구로 '기상 변수 통합 순환 신경망을 활용한
제주시 전력 수요 예측' 논문이 발표되어 있습니다."
```

## 5. 부록

### 5.1 기술 스펙 요약

```yaml
모델:
  - LSTM, BiLSTM, TFT 앙상블
  - Quantile Regression (10%, 50%, 90%)
  - 파라미터: 171,890개 (경량화)

성능:
  - MAPE: 10.68%
  - MAE: 11.27 원/kWh
  - 80% Coverage: 82.5%

데이터:
  - 학습 기간: 2022-2024 (26,240건)
  - 검증: Walk-forward 5-fold CV

인프라:
  - API: FastAPI REST
  - Dashboard: Streamlit
  - 테스트: 1,436건 통과
```

### 5.2 경쟁 우위

1. **학술적 검증**: 논문 발표 (정현수, 길준민, 2025)
2. **실제 데이터 검증**: EPSIS 실측 데이터 사용
3. **설명 가능성**: XAI 기반 투명한 예측
4. **불확실성 정량화**: Quantile Regression
